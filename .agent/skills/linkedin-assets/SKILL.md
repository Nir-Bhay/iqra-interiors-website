# LinkedIn Assets Skill

This skill defines the rigorous standards for creating "Premium" LinkedIn assets.

## 🎨 Design Philosophy: The "Nana Banan" Standard
(Premium, Aesthetic, Clean, Modern, Professional)

1.  **Aspect Ratio Rule**: ALWAYS 4:1 (Target: 1584 x 396 px).
2.  **The "No-Clutter" Rule**: Banners sit *behind* the profile photo. The bottom-left corner (~200px) is a "Dead Zone". Keep it empty.
3.  **Text Minimalism**: Avoid heavy text. Let the headline do the talking. The banner is for *vibe* and *branding*.
4.  **Premium Aesthetics**:
    - **Dark Mode**: Matte blacks, deep charcoals, slate greys > Pure black.
    - **Accents**: One neon/glowing color (Cyan, Electric Purple, Gold) to pop.
    - **Texture**: Subtle noise, grain, or gradients to prevent "flat" cheap looks.
