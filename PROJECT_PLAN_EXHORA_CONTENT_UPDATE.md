## ❌ DON'Ts (STRICTLY AVOID)

### Absolutely NO Changes To:
1. ❌ **HTML Structure** - No adding/removing tags, divs, sections
2. ❌ **CSS Files** - Zero modifications to any .css files
3. ❌ **CSS Classes/IDs** - Keep all class names unchanged
4. ❌ **JavaScript** - No changes to .js files or script tags
5. ❌ **Images** - Don't change image paths or attributes (image replacement is separate task)
6. ❌ **Layout** - No repositioning of elements
7. ❌ **HTML Attributes** - Keep data-*, aria-*, role attributes as-is
8. ❌ **Color Schemes** - Not part of content update (client mentioned separately)
9. ❌ **UI Components** - No changes to buttons, forms, navigation structure
10. ❌ **Animations/Effects** - Keep all existing functionality

### Content Don'ts:
1. ❌ Don't copy-paste content directly from reference websites (plagiarism)
2. ❌ Don't use generic template content
3. ❌ Don't change brand name "Exhora Ambiance" (note spelling)
4. ❌ Don't add fictional claims or fake testimonials
5. ❌ Don't exceed character limits that would break design
6. ❌ Don't use overly technical jargon that alienates clients
